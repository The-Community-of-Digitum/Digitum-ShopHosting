In here you may shop freely.
Find a folder, buy what you want!
Maybe strike up a trade, if you're brave.

This repository hosts all official shops of **The Republic of Digitum**.

Each shop lives in its own folder and contains:
- a shop file (`shop.md`)
- item listings
- pricing
- descriptions

Citizens (or higher) may open their own shops by creating a new folder and submitting a PR.

## How to Buy

To purchase an item from any shop:

1. Open a Pull Request in this repository.
2. Include:
   - your name
   - the shop name
   - the item you want
   - the price
3. The shop owner will approve the PR.
4. The ledger will be updated with the D$ transfer.
5. You receive your item via a comment on the shop.
6. SAY A COMMENT!
7. The two comments will then be deleted to prevent stealing via spying.
